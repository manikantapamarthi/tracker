  `Rails 7.0.8`

To run this application in your local environment 

clone app 

`https://github.com/manikantapamarthi/tracker.git`

create the database 

`rails db:create`

to run the migrations 

`rails db:migrate`

to run the seeds 
`rails db:seed`
